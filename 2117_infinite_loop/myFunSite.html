What We Do

We are basically learn something of social network, information, and technology. In each topic, we have several modules to do. Moreover, students have to finish Learner, Thinker, Speaker, Writer, and Builder to get leveled up and then get 4.0. In order to get 4.0, students have to get total 13405XP at 100% quality. Of each percent of quality down, students need to get extra corrosponding XP to get 4.0. Here is out class link: gclass.

Learner

Students will do 25 modules in Learner. In each module, students will watch a video and fill in the blank alongside the video. Student have to finish the current module to unlock the following module.

Thinker

Thinker has quize, face-off, and learn and teach section. After finish the modules in the Learner, students are allowed to do the quiz in Thinker, which corrosponds to the module in Learner, but the quiz section has to be done during the class time. After finish the quiz, students can do face-off to compete with others. If they win, they can do the Learn and Teach section.

Speaker

Student also need to do Quiz and Face-off sections in the Spearker, but the difference is that student will write a story by using some academic words.

Builder

We will learn some code techniques in Builder, such as HTML, CSS, and Databases. TA will teach students how to write a HTML file or how to use CSS to change HTML web page. After that, student will have some homework to practice what they learned in class.

TA

TA

Teaching Assistant is the "main teacher" of the INFO 101 because students will have classes with TA twice a week, and they will have a 1h 50mins class with TA each time, but they will have 50mins class with Professor Bob once a week.
